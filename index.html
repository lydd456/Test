Shopping Guide

Asking Price

duō shǎo = how much?
duō shǎo qián = how much money?

Money in Chinese

Informal = yī kuài (一块)
Formal = yī yuán (一元)

Numbers in Chinese

yī (一), èr (二) / liǎng (两) , sān (三), sì (四), wǔ (五), liù (六), qī (七), bā (八), jiǔ (九), shí (十) = 1-10
yī bǎi (一百) = 100
yī qiān (一千) = 1000
yī wàn (一万) = 10000

Buying in Chinese

Wǒ xiǎng mǎi zhège = I want to buy this
Yǒu dǎzhé ma? (有打折吗?) = Is there discount?

Negotiation in Chinese

Kěyǐ shí kuài qián ma? (可以十块钱吗?) = Can it be 10 yuan?
Tài guì le (太贵了) = Too expensive
Kěyǐ piányi yīdiǎn ma? (可以便宜一点吗) = Can it be a little cheaper?

Paying in Chinese

Kěyǐ fù xiànjīn? (可以付现金?) = Can pay using cash?
Kěyǐ fù Zhīfùbǎo? (可以付支付宝) = Can pay using alipay?
Kěyǐ fù Visa kǎ ma? (可以付 Visa 卡吗？) = Can pay using visa card?

Restaurant Guide

Intro

Qǐngwèn, yǒu càidān ma? = Excuse me, do you have a menu?
Nǐmen yǒu jǐ ge rén? / Nǐmen yǒu duōshǎo rén? = How many people are you?
Wǒmen sān ge rén = We are 3 people

How to read the menu

鸡 (jī) = chicken
牛 (niú) = cow
鱼 (yú) = fish
羊 (yáng) = lamb
肉 (ròu) = meat
米饭 (mǐfàn) = rice
炒饭 (chǎo fàn) = fried rice
面(条) (miàn(tiáo)) = noodles (on menu usually just written mian and the variation of noodle e.g. lamian)
蛋 (dàn) = egg

Paying in restaurant

Wǒ xiǎng fù kuǎn = I want to pay
Zǒnggòng (Yígòng) duōshǎo qián? = How much is the total?
Kěyǐ fù xiànjīn? = Can pay using cash?
Kěyǐ fù Zhīfùbǎo? = Can pay using alipay?
Kěyǐ fù Visa kǎ ma? = Can pay using visa card?
